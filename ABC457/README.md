## C - Long Sequence

给定整数 N, K，N 个整数列 A1, A2, ..., AN，以及长度 N 的整数列 C。

Ai 的长度为 Li，即 Ai = (Ai,1, Ai,2, ..., Ai,Li)。

按照以下步骤构造整数列 B：
- B 初始为空。
- 依次对 i = 1, 2, ..., N：将 Ai 拼接到 B 的末尾，重复 Ci 次。

求 B 的第 K 个元素 BK 的值。

**约束:**
- 1 ≤ N
- 1 ≤ Li
- ΣLi ≤ 2×10^5
- 1 ≤ Ai,j ≤ 10^9
- 1 ≤ Ci ≤ 10^9
- 1 ≤ K ≤ Σ Ci×Li

**输入:**
```
N K
L1 A1,1 A1,2 ... A1,L1
L2 A2,1 A2,2 ... A2,L2
...
LN AN,1 AN,2 ... AN,LN
C1 C2 ... CN
```

**输出:**
BK 的值

## D - Raise Minimum

给定长度为 N 的数列 A = (A1, A2, ..., AN) 和整数 K。

你可以进行 0 次以上、K 次以下的操作：
- 选择满足 1 ≤ i ≤ N 的整数 i，将 Ai 加上 i。

求操作后数列的最小值 min(Ai) 可能达到的最大值。

**约束:**
- 1 ≤ N ≤ 2×10^5
- 1 ≤ Ai ≤ 10^18
- 1 ≤ K ≤ 10^18

**输入:**
```
N K
A1 A2 ... AN
```

**输出:**
最小值的最大值
